# 🛒 E-Commerce Web Application

A full-stack E-Commerce web application built using **Spring Boot**, **React.js**, and **MySQL**. The application allows customers to browse products, manage their shopping cart, place orders, and enables administrators to manage products and users through a secure dashboard.

---

## 🚀 Features

### Customer

* User Registration & Login
* JWT Authentication
* Browse Products by Category
* Search Products
* Add to Cart
* Place Orders
* View Order History

### Admin

* Secure Admin Login
* Add, Update, and Delete Products
* Manage Product Categories
* View Customer Orders
* Role-Based Access Control

## ▶️ How to Run

### Backend

1. Clone the repository.
2. Configure the MySQL database credentials.
3. Run the Spring Boot application.
4. Access the backend at:

   ```
   http://localhost:8080
   ```

### Frontend

1. Navigate to the frontend directory.
2. Install dependencies:

   ```bash
   npm install
   ```
3. Start the React application:

   ```bash
   npm start
   ```
4. Access the frontend at:

   ```
   http://localhost:3000
   ```
